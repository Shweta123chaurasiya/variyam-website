# VARIYAM 🌿

VARIYAM is an impact-driven community immersion and rural tourism platform based in Varanasi, India.

The project focuses on creating meaningful travel experiences that connect travelers, students, and organizations with rural communities, traditions, culture, sustainability, and local livelihoods.

---

## 🌍 Vision

To build bridges between urban travelers and rural communities through immersive cultural experiences, learning journeys, and sustainable tourism.

---
